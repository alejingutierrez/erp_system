# DropdownSelect

The `DropdownSelect` component provides a styled dropdown menu for selecting one or more options. It can include a search field when the option list is long.

<Canvas>
  <Story name="Example">
    {() => {
      const [value, setValue] = React.useState('');
      return (
        <div style={{ minHeight: '6rem' }}>
          <DropdownSelect
            options={["S", "M", "L", "XL"]}
            selected={value}
            onChange={setValue}
            placeholder="Seleccione"
          />
        </div>
      );
    }}
  </Story>
</Canvas>
