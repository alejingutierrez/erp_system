# GlobalHeader

The `GlobalHeader` organizes primary navigation, search and user actions in a single top bar. It collapses the navigation items into a menu button on narrow screens.

## Responsive behavior

Below the `sm` breakpoint the navigation items are hidden and can be accessed through a menu toggle. Use the `onNavigate`, `onSearch` and `onNotificationsOpen` callbacks to capture user actions.
