# GlobalHeader

The `GlobalHeader` combines a fixed top bar with a collapsible sidebar. It holds universal search, multi‑store selector, notifications, help menu and profile. Keyboard shortcuts (`⌘/Ctrl+K` and `⌘/Ctrl+Shift+N`) are available by default.

## Responsive behavior

Below the desktop breakpoint (`1024px`), the sidebar collapses into a menu button and the search bar moves into the drawer. Use the callbacks to hook into navigation, search and notifications.
